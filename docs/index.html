PIDron Desktop

Локальний застосунок для емулювання сетапу дрона, прозорої фізики, підбору PID, пресетів, 3D-візуалізації, Blackbox/CSV аналізу та роботи з Betaflight, INAV і ArduPilot.

Що вміє

Setup simulator5/7/10/custom, quad/hex/octo/coax, mixed pusher motors, custom props and batteries.
Transparent physicsThrust, inertia, motor lag, battery sag, wind, gyro noise, prop loading and saturation are shown as understandable metrics.
PID tuningRoll/pitch/yaw rate-loop simulation with P/I/D/feedforward recommendations, overshoot, settling and oscillation risk.
PresetsSave named builds locally, reopen them from a dedicated Presets page, and keep setup experiments organized.
Flight controller labGenerate Betaflight/INAV CLI or ArduPilot params, read PID text from serial, and write with an explicit safety gate.
AnalysisCSV/TXT log analysis MVP with honest binary .bbl detection and a clear path toward a native parser.
